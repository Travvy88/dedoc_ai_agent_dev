\par \b Module Contract\/ \:\par \b Purpose\/ \: Define the Table schema — a Pydantic model for a document table with row-based cell grid and table-level metadata.

\par \b Scope\/ \: Document table data model. \par \b Input\/ \: None (standalone model, references CellWithMeta, TableMetadata). \par \b Output\/ \: Pydantic BaseModel Table with cells (List[List[CellWithMeta]]) and metadata (TableMetadata). \par \b Links\/ \: [USES_API(7): pydantic.BaseModel; READS_DATA_FROM(8): CellWithMeta, TableMetadata] \par \b Invariants\/ \:

  • cells grid is rectangular (all rows have equal column count).
  • metadata.uid uniquely identifies the table within the document. \par \b Rationale\/ \:

Q: Why row-based cell storage? A: Matches natural document table layout and simplifies HTML rendering (iterate rows, then cells). \par \b Changes\/ \:
