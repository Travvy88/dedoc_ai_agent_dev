\par \b Module Contract\/ \:\par \b Purpose\/ \: To normalize xlsx-like documents (.xls, .ods) into the XLSX format via LibreOffice headless conversion, providing a uniform input format for downstream spreadsheet readers.

\par \b Scope\/ \: Excel-like document format normalization via soffice. \par \b Input\/ \: Files with excel-like extensions or MIME types. \par \b Output\/ \: Converted .xlsx file path. \par \b Links\/ \: [USES_API(9): LibreOffice soffice; READS_DATA_FROM(7): dedoc.extensions.converted_extensions] \par \b Invariants\/ \:

  • convert() always invokes soffice with –headless –convert-to xlsx.
  • Output file path is always {dir}/{name_wo_ext}.xlsx. \par \b Rationale\/ \:

Q: Why soffice instead of openpyxl for conversion? A: soffice handles legacy binary .xls and .ods formats that openpyxl cannot read. \par \b Changes\/ \:
