\par \b Module Contract\/ \:\par \b Purpose\/ \: Define the base Annotation class — the root of the annotation hierarchy for all text-level metadata in the document processing pipeline.

\par \b Scope\/ \: Base annotation class with position, value, equality, serialization. \par \b Input\/ \: Start/end positions, annotation name, value string. \par \b Output\/ \: Annotation base instances consumed by all annotation subclasses. \par \b Links\/ \: [EXPORTS(10): Annotation, INHERITS(4): Serializable, USES_API(8): ApiAnnotation] \par \b Invariants\/ \:

  • start <= end (not enforced but expected by convention)
  • name is a non-empty string unique per annotation type
  • is_mergeable defaults to True \par \b Rationale\/ \:

Q: Why inherit from Serializable? A: All annotations must be convertible to API schemas for HTTP responses. \par \b Changes\/ \:
