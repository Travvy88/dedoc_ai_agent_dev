\par \b Module Contract\/ \:\par \b Purpose\/ \: To normalize docx-like documents (.doc, .rtf, .odt) into the DOCX format via LibreOffice headless conversion, providing a uniform input format for downstream readers.

\par \b Scope\/ \: Docx-like document format normalization via soffice. \par \b Input\/ \: Files with docx-like extensions or MIME types. \par \b Output\/ \: Converted .docx file path. \par \b Links\/ \: [USES_API(9): LibreOffice soffice; READS_DATA_FROM(7): dedoc.extensions.converted_extensions] \par \b Invariants\/ \:

  • convert() always invokes soffice with –headless –convert-to docx.
  • Output file path is always {dir}/{name_wo_ext}.docx. \par \b Rationale\/ \:

Q: Why soffice instead of python-docx for conversion? A: soffice handles legacy binary .doc and .rtf formats that python-docx cannot read. It is the single most reliable tool for format normalization. \par \b Changes\/ \:
