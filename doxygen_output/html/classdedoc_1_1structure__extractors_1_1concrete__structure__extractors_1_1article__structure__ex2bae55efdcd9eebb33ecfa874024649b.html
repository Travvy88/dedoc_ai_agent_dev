Detailed Description

\par \b Purpose\/ \: ArticleStructureExtractor for document structure extraction pipeline

This class corresponds to the GROBID <https://grobid.readthedocs.io/en/latest/>_ article structure extraction.

This class saves all tag_hierarchy_levels received from the :class:~dedoc.readers.ArticleReader \ without using the postprocessing step (without using regular expressions).

You can find the description of this type of structure in the section :ref:article_structure.
