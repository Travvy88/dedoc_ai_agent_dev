Detailed Description

\par \b Purpose\/ \: Base class for all text annotations — holds start/end positions, name, value, and mergeability flag for text region metadata.

Base class for text annotations of all kinds. Annotation is the piece of information about the text line: it's appearance or links to another document object. Look to the concrete kind of annotations to get mode examples.

:ivar start: start of the annotated text :ivar end: end of the annotated text (end isn't included) :ivar name: annotation's name, specific for each type of annotation :ivar value: information about annotated text, depends on the type of annotation, e.g. "True"/"False", "10.0", etc. :ivar is_mergeable: is it possible to merge annotations with the same value

:vartype start: int :vartype end: int :vartype name: str :vartype value: str :vartype is_mergeable: bool

\par \b Purpose\/ \: Initialize annotation with position, name, value, and mergeability flag.

\par \b Uses\/ \: Serializable \par \b I/O\/ \: (int, int, str, str, bool) -> None \par \b Complexity\/ \: 2

Some kind of text information about symbols between start and end. For example Annotation(1, 13, "italic", "True") says that text between 1st and 13th symbol was writen in italic.

:param start: start of the annotated text :param end: end of the annotated text (end isn't included) :param name: annotation's name :param value: information about annotated text :param is_mergeable: is it possible to merge annotations with the same value
