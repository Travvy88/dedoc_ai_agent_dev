Detailed Description

\par \b Purpose\/ \: Compute numbering text for ordered list items (arabic, alphabetic, roman numerals) according to PowerPoint ST_TextAutonumberScheme.

\par \b Uses\/ \: None (pure computation class) \par \b Complexity\/ \: 4

This class is used to compute numbering text for list items. For example: "1.", (i), "○"
