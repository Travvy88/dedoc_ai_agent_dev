\par \b Module Contract\/ \:\par \b Purpose\/ \: To provide utilities for preparing training datasets from document processing results: saving LineWithMeta objects as JSON lines, merging lines with same BBox, and managing image/document paths for ML model training workflows.

\par \b Scope\/ \: Training dataset creation, line postprocessing (BBox deduplication), image path management, document path resolution. \par \b Input\/ \: LineWithMeta objects, config dict, original document path. \par \b Output\/ \: JSON lines files, image directories, document paths. \par \b Links\/ \: [USES_API(6): PIL, numpy, json] \par \b Invariants\/ \:

  • __postprocess_lines merges consecutive lines with identical bounding boxes.
  • save_line_with_meta always creates intermediate directories if they do not exist. \par \b Rationale\/ \:

Q: Why merge lines with identical BBox? A: Some readers produce multiple LineWithMeta fragments for the same visual region. Pre-merging simplifies annotation and reduces dataset noise. \par \b Changes\/ \:
