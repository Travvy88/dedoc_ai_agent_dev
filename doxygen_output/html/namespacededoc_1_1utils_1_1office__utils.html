\par \b Module Contract\/ \:\par \b Purpose\/ \: To extract and parse XML content from Office Open XML format files (docx, pptx, xlsx) via zipfile + BeautifulSoup, serving as a shared utility for Office format readers.

\par \b Scope\/ \: ZIP archive XML extraction, whitespace normalization, BeautifulSoup tree construction. \par \b Input\/ \: Path to office format file, internal XML path. \par \b Output\/ \: BeautifulSoup XML tree or None if the file is not found inside the archive. \par \b Links\/ \: [USES_API(7): zipfile, BeautifulSoup] \par \b Links to Spec\/ \: REQ-OFFICE-001 \par \b Invariants\/ \:

  • Returns None for missing internal XML paths without raising.
  • Raises BadFileFormatError on corrupted ZIP archives. \par \b Rationale\/ \:

Q: Why remove whitespace between XML tags? A: Office XML files often contain formatting whitespace that interferes with BeautifulSoup parsing. Pre-cleaning yields cleaner parse trees. \par \b Changes\/ \:
