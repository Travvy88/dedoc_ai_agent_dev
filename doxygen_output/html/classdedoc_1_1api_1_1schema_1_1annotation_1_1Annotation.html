Detailed Description

\par \b Purpose\/ \: Represent a single inline annotation (e.g., bold, italic, link) spanning a character range within document text.

\par \b I/O\/ \: (int, int, str, str) -> JSON serializable model

The piece of information about the text line: it's appearance or links to another document object. For example Annotation(1, 13, "italic", "True") says that text between 1st and 13th symbol was written in italic.

:ivar start: start of the annotated text :ivar end: end of the annotated text (end isn't included) :ivar name: annotation's name, specific for each type of annotation :ivar value: information about annotated text, depends on the type of annotation, e.g. "True"/"False", "10.0", etc.

:vartype start: int :vartype end: int :vartype name: str :vartype value: str
