Detailed Description

This class allows to extract content (textual and table) from the .pdf documents with a textual layer (copyable documents).

It uses java code to get the result.

It is recommended to use this class as a handler for PDF documents with a correct textual layer if you don't need to check textual layer correctness. For more information, look to pdf_with_text_layer option description in :ref:pdf_handling_parameters.
