\par \b Module Contract\/ \:\par \b Purpose\/ \: Define the DocumentContent data structure — the primary container for a document's parsed text (as a tree) and tables.

\par \b Scope\/ \: Structured document content: text tree, tables, warnings. \par \b Input\/ \: Tables list, TreeNode root, optional warnings. \par \b Output\/ \: DocumentContent instance convertible to API schema. \par \b Links\/ \: [INHERITS(5): Serializable, USES_API(8): ApiDocumentContent, READS_DATA_FROM(9): Table, TreeNode] \par \b Invariants\/ \:
