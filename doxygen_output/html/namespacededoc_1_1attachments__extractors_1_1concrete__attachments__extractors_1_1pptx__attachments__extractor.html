\par \b Module Contract\/ \:\par \b Purpose\/ \: Extract embedded attachments (images, OLE objects) from PowerPoint (.pptx) files via ZIP+OLE introspection.

\par \b Scope\/ \: pptx file attachment extraction: reads zip archive, locates media/embeddings in ppt/ folder, parses OLE streams. \par \b Input\/ \: File path to .pptx document; optional parameters dict (with_attachments, need_content_analysis, attachments_dir). \par \b Output\/ \: List of AttachedFile objects extracted from the pptx archive. \par \b Links\/ \: [USES_API(9): AbstractOfficeAttachmentsExtractor._get_attachments] \par \b Links\/ \: [USES_API(8): recognized_extensions.pptx_like_format, recognized_mimes.pptx_like_format] \par \b Invariants\/ \:

  • extract ALWAYS returns a List[AttachedFile] (may be empty).
  • Parameters dict defaults to {} if None is passed. \par \b Rationale\/ \:

Q: Why delegate to _get_attachments with attachments_dir="ppt"? A: pptx is an OpenXML format; the media folder resides under ppt/ in the zip. The base class handles the zip iteration and OLE parsing generically. \par \b Changes\/ \:
