\par \b Module Contract\/ \:\par \b Purpose\/ \: Extract embedded attachments (images, charts, OLE objects) from Excel (.xlsx) files via ZIP+OLE introspection.

\par \b Scope\/ \: xlsx file attachment extraction: reads zip archive, locates media/embeddings in xl/ folder, parses OLE streams. \par \b Input\/ \: File path to .xlsx document; optional parameters dict (with_attachments, need_content_analysis, attachments_dir). \par \b Output\/ \: List of AttachedFile objects extracted from the xlsx archive. \par \b Links\/ \: [USES_API(9): AbstractOfficeAttachmentsExtractor._get_attachments] \par \b Links\/ \: [USES_API(8): recognized_extensions.excel_like_format, recognized_mimes.excel_like_format] \par \b Invariants\/ \:

  • extract ALWAYS returns a List[AttachedFile] (may be empty).
  • Parameters dict defaults to {} if None is passed. \par \b Rationale\/ \:

Q: Why delegate to _get_attachments with attachments_dir="xl"? A: xlsx is an OpenXML format; the media folder resides under xl/ in the zip. The base class handles the zip iteration and OLE parsing generically. \par \b Changes\/ \:
