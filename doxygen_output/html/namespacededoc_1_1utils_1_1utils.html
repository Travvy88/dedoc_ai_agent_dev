\par \b Module Contract\/ \:\par \b Purpose\/ \: To provide general-purpose utility functions for file handling, MIME type detection, encoding detection, text similarity, and datetime conversion across the Dedoc document processing pipeline.

\par \b Scope\/ \: File I/O, MIME type inference, encoding detection, string similarity, datetime parsing, file hashing, batch iteration. \par \b Input\/ \: File paths, binary data, text strings, UploadFile objects. \par \b Output\/ \: File paths, MIME types, encodings, hashes, similarity scores, Unix timestamps. \par \b Links\/ \: [USES_API(7): mimetypes, hashlib, charset_normalizer, Levenshtein, requests] \par \b Invariants\/ \:

  • get_file_mime_type ALWAYS returns a non-empty string.
  • calculate_file_hash ALWAYS returns a hex digest string.
  • splitext_ correctly handles multi-dot extensions (e.g., .tar.gz). \par \b Rationale\/ \:

Q: Why use charset_normalizer instead of chardet? A: charset_normalizer provides more accurate encoding detection and is actively maintained. \par \b Changes\/ \:
