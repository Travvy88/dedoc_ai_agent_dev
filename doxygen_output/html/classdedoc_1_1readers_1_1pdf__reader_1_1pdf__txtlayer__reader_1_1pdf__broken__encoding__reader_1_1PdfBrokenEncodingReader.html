Detailed Description

This class allows to extract content (text, tables, attachments) from the .pdf documents with a textual layer with broken encoding (copyable documents, but copied text is incorrect) with complex background.

It uses a pdfminer library for text extraction and CNN for font's glyphs prediction. Currently, only Russian and English languages are supported.

For more information, look to pdf_with_text_layer option description in :ref:pdf_handling_parameters.
