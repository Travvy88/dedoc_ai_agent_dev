Detailed Description

\par \b Purpose\/ \: Hold table-level metadata: unique identifier (for cross-referencing), page number, rotation angle, and optional title.

\par \b I/O\/ \: (page_id, uid, rotated_angle, title) -> JSON serializable model

Holds the information about table unique identifier, rotation angle (if table has been rotated - for images) and so on.

:ivar page_id: number of the page where table starts :ivar uid: unique identifier of the table (used for linking table to text) :ivar rotated_angle: value of the rotation angle by which the table was rotated during recognition :ivar title: table's title

:vartype page_id: Optional[int] :vartype uid: str :vartype rotated_angle: float :vartype title: str
