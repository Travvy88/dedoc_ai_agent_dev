\par \b Module Contract\/ \:\par \b Purpose\/ \: To normalize pdf-like documents (.djvu) into the PDF format via the ddjvu utility, providing a uniform input format for downstream PDF readers.

\par \b Scope\/ \: PDF-like document format normalization via ddjvu. \par \b Input\/ \: Files with pdf-like extensions or MIME types (e.g. .djvu). \par \b Output\/ \: Converted .pdf file path. \par \b Links\/ \: [USES_API(9): ddjvu (DjVuLibre); READS_DATA_FROM(7): dedoc.extensions.converted_extensions] \par \b Invariants\/ \:

  • convert() always invokes ddjvu with –format=pdf.
  • Output file path is always {dir}/{name_wo_ext}.pdf. \par \b Rationale\/ \:

Q: Why ddjvu instead of a Python library? A: ddjvu is the reference DjVu-to-PDF converter from the DjVuLibre project. No pure-Python equivalent exists. \par \b Changes\/ \:
