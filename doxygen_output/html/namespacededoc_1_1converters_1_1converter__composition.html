\par \b Module Contract\/ \:\par \b Purpose\/ \: To provide a composite converter that iterates through a configured chain of format-specific converters, using the first match to transform any document into a processable format.

\par \b Scope\/ \: Converter chain orchestration, MIME detection delegation, file permission hardening. \par \b Input\/ \: converters: List[AbstractConverter] passed at construction. \par \b Output\/ \: Converted file path (str). \par \b Links\/ \: [USES_API(8): dedoc.utils.utils.get_mime_extension, dedoc.converters.AbstractConverter] \par \b Invariants\/ \:

  • convert() always returns a valid file path (original or converted).
  • Output file is always made world-readable via chmod. \par \b Rationale\/ \:

Q: Why a chain pattern instead of a registry? A: Converter ordering matters (e.g., DOCX before generic binary). A list preserves user-specified priority. \par \b Changes\/ \:
