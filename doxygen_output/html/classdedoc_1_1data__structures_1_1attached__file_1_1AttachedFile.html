Detailed Description

\par \b Purpose\/ \: Hold information about files attached to the parsed document: original name, disk path, content analysis flag, and UID.

Holds information about files, attached to the parsed document.

:ivar original_name: original name of the attached file if it was possible to extract it :ivar tmp_file_path: path to the attached file on disk - its name is different from original_name :ivar need_content_analysis: does the attached file need parsing (enable recursive parsing in :class:~dedoc.DedocManager) :ivar uid: unique identifier of the attached file

:vartype original_name: str :vartype tmp_file_path: str :vartype need_content_analysis: bool :vartype uid: str
